## 📖 Overview

Ghar Sewa is a web-based home service platform designed to bridge the gap between customers and service providers. The system enables users to book household services, manage requests, and monitor service progress through an intuitive and secure interface.

---

## ✨ Key Features

✅ User Registration & Authentication

✅ Service Booking & Scheduling

✅ Service Provider Management

✅ Request Tracking System

✅ Admin Dashboard

✅ Customer Profile Management

✅ Responsive User Interface

✅ Secure Session-Based Login

## 📊 System Modules

### 👤 User Module
- Register and Login
- Browse Available Services
- Book Home Services
- Track Service Requests

### 🔧 Service Provider Module
- Manage Assigned Tasks
- Update Service Status
- View Customer Requests

### 🛡️ Admin Module
- Manage Users
- Manage Service Providers
- Monitor Bookings
- Generate Reports

---

## 🎯 Project Objective

To develop a centralized platform that simplifies the process of finding, booking, and managing home services while ensuring convenience, reliability, and efficiency for both customers and service providers.
